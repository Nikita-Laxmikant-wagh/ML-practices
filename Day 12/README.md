# House Price Prediction using Linear Regression

## Project Overview

This project predicts house prices using Machine Learning. A Linear Regression model is trained on historical housing data and predicts the selling price of a house based on its features.

## Dataset Features

* MSSubClass
* MSZoning
* LotArea
* LotConfig
* BldgType
* OverallCond
* YearBuilt
* YearRemodAdd
* Exterior1st
* BsmtFinSF2
* TotalBsmtSF

## Target Variable

* SalePrice

## Project Workflow

1. Data Loading
2. Data Understanding
3. Data Cleaning
4. Handling Missing Values
5. Encoding Categorical Features
6. Feature Scaling
7. Train-Test Split
8. Linear Regression Model Training
9. Prediction
10. Model Evaluation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Evaluation Metrics

* R² Score
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

## Conclusion

The Linear Regression model successfully predicts house prices based on property characteristics and can help automate price estimation for real estate businesses.
